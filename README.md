# Calculator App

A simple calculator application built using HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations (+, -, *, /)
- Clear button (C)
- Backspace button (⌫)
- Error handling for invalid expressions
- Responsive calculator layout
- Interactive button effects

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Project Structure

```text
calculator-app/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How It Works

1. User clicks calculator buttons.
2. Input is stored in a JavaScript variable called `expression`.
3. The display updates in real time.
4. When `=` is pressed, the expression is evaluated and the result is shown.
5. Invalid expressions are handled using `try...catch`.
